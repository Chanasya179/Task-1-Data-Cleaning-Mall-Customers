## Task 1: Data Cleaning and Preprocessing

## Objective
Clean and preprocess raw Mall Customers dataset using Python (Pandas).

## Dataset
- Source: Kaggle - Mall Customer Segmentation Data
- Rows: 200 | Columns: 5

## Steps Performed

1. Loaded dataset using Pandas
2. Checked missing values using isnull()
3. Verified no missing values existed
4. Checked duplicate records
5. Verified no duplicate rows existed
6. Standardized Gender column values
7. Renamed column headers
8. Checked data types
9. Saved cleaned dataset

## Files
- Mall_Customers.csv — Original raw dataset
- mall_customers_cleaned.csv — Cleaned dataset
- MallSegmentation.ipynb — Python notebook with all cleaning steps

## Tools Used

- Python
- Pandas
- Google Colab

## Result

The dataset was cleaned, standardized, and prepared for analysis.
